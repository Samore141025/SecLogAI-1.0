# Security Policy

## Supported Versions

We currently support the latest version of SecLogAI.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of SecLogAI seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not open a public issue.**
2. Email the details of the vulnerability to `security@example.com` (replace with your actual contact).
3. Include as much information as possible:
   - Type of issue (e.g., XSS, SQLi, etc.)
   - Location of the vulnerability
   - Steps to reproduce
   - Potential impact

We will acknowledge your report within 48 hours and provide a timeline for a fix. We ask that you practice responsible disclosure and give us time to address the issue before making it public.
